Part one, Task Overview.

Now, let’s move to the motivation of our work. Clustering is a core challenge in unsupervised ML, and traditional methods like K-means struggle with high-dimensional data due to the curse of dimensionality. As shown in Figure 1, even with deep networks, data categories often overlap before clustering—leading to poor cluster purity. Advanced methods like SCCL and DACL can separate clusters to some extent, but still fail to form clear, distinct cluster boundaries. That’s the problem we aim to address.

Now, let's introduce our proposed method. We design a joint model that integrates instance-wise contrastive learning (Instance-CL) with clustering for unsupervised representation learning. As shown in Figure 2, a backbone ψ(·) maps input data to embeddings, then two heads follow: g(·) for contrastive loss, and f(·) for clustering loss. Our data includes both original and augmented samples: for a minibatch B, we randomly generate augmented data B^a.

Now let’s look at Figure 2—this is our proposed ISCST framework: it combines bottom-up instance discrimination with top-down clustering objectives, built on Sentence-Transformers plus contrastive learning. ISCST uses contrastive learning to boost categorical separation, while optimizing intra-cluster compactness and inter-cluster discrimination. That’s how our model works.

Part three, Experimental Method.

Now, let’s talk about implementation details: We built ISCST using PyTorch and the Sentence Transformer library. For embeddings, we used the all-mpnet-base-v2 model. The clustering head (f) is a linear layer with 768×K dimensions (K = number of clusters), and the contrastive learning component uses an MLP (g) with 768 hidden units and 128-dimensional outputs.

Now let’s cover our data augmentation exploration—we tested 4 unsupervised strategies: First, Word Deletion (randomly removes words via WordNet); second, Charswap (swaps random characters); third, Contextual Augmenter (uses BERT/RoBERTa for semantic-based insertion/substitution); fourth, Paraphrase Augmenter (round-trip translation via English-French models). These strategies help generate robust positive pairs for contrastive learning.

Let’s take a look at Figure 3—it shows how our paraphrase augmentation works. We start with an example input sentence about the Games’ postponement. Using round-trip translation (English-French-English) plus a mixture-of-experts model, we generate two paraphrased outputs, as shown here. This is how we create augmented text pairs for our model.

Part four, Experimental Result.

Now, let's talk about evaluation metrics: we use four standard ones—ACC (matches predicted and true categories), NMI (information correlation between results and labels), ARI (consistency adjusted for chance), and AMI (NMI with random clustering correction). For short text clustering, we focus on ACC and NMI: they offer complementary views via direct category matching and information-theoretic relationships.

Now we’ll compare ISCST to state-of-the-art short text clustering methods—like STCC (three-stage optimization), Self-Train (autoencoder-based), SimCTC (simple contrastive learning), and others like SCCL and DACL. Our method updates target distributions at each iteration. This lets it outperform multi-stage and hierarchical approaches, achieving superior performance overall.

Our method updates target distributions each iteration—and this helps it outperform these multi-stage/hierarchical approaches, showing better overall performance.
